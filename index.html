Setup Guide
Go live in 2 minutes
VoxPath works instantly with zero setup. Steps below are all optional upgrades.
1
Just click "Call Siya now"
The portal works out of the box. Siya uses a built-in conversation engine. No API key needed. Works completely offline.
2
Allow microphone in Chrome
Chrome will ask for microphone permission the first time. Click Allow. That's it — STT and TTS are built into the browser.
3
(Optional) Get free Groq key for smarter AI
Go to console.groq.com → sign up free → create API key → paste in Integrations. Unlocks Llama 3.3 70B: much more natural Hinglish conversation.
Open Groq Console
4
(Optional) Connect Google Sheets
Auto-log call transcripts and lead scores to a Google Sheet via Apps Script webhook.
5
Use Chrome or Edge only
The Web Speech API is only available in Chromium-based browsers. Safari and Firefox won't work for voice calls.
