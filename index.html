Setup guide
VoxPath works instantly. All steps below are optional upgrades.
1
Click "Call Siya now" — it works immediately
The built-in conversation engine requires zero configuration. Works offline. No API key needed.
2
Allow microphone in Chrome or Edge
The browser will ask for microphone permission on the first call. Click Allow. Speech-to-text and text-to-speech are built into the browser — no extra setup.
3
(Optional) Get a free Groq key for smarter AI
Go to console.groq.com → sign up free → create API key → paste in Integrations. Unlocks Llama 3.3 70B for much more natural Hinglish conversation. 14,400 requests/day free.
Open Groq Console
4
(Optional) Get a free ElevenLabs key for natural voice
Go to elevenlabs.io → sign up free → Profile → API Keys → Create → paste in Integrations. 10,000 characters/month free. No credit card required.
Open ElevenLabs
5
Chrome or Edge only for voice calls
The Web Speech API used for speech recognition is only available in Chromium-based browsers. Safari and Firefox are not supported.
