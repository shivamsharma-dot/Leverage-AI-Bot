Setup guide
7 steps to your first live call.
Everything runs in a browser. No Python or Node.js needed on your machine.
Create a GitHub account
Sign up at github.com and enable Codespaces — you get a full browser-based VS Code with terminal, 60 hrs/month free. This is your entire development environment.
100% free
2
Get your Deepgram API key
deepgram.com → sign up → copy API key. Converts your caller's voice to text in real-time. 12,000 minutes per year free — enough for months of testing.
12K mins/yr
3
Get Gemini Flash API key
aistudio.google.com → Get API key. Gemini 1.5 Flash is completely free, responds under 400ms, handles complex conversations naturally.
Free forever
4
Enable Google Cloud TTS
console.cloud.google.com → enable Text-to-Speech API. Download credentials JSON. WaveNet voices — 1 million characters per month free.
1M chars/mo
5
Fork the VoxPath starter repo
One click on GitHub — copies the entire backend codebase into your account. Open in Codespaces and you have a working editor with all dependencies ready.
Browser only
6
Add secrets and deploy to Cloud Run
Repo → Settings → Secrets → add DEEPGRAM_KEY, GEMINI_KEY, GOOGLE_TTS_JSON. Push once — GitHub Actions builds and deploys automatically.
20 mins
7
Connect a phone number via Vapi.ai
Vapi.ai free tier gives you a real phone number. Paste your Cloud Run URL as the webhook. Call the number — your AI answers.
Vapi free
