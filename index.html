Setup Guide
Go live in 15 minutes
Everything you need to launch your first AI outbound call. Completely free.
1
Get a free Gemini API key
Go to aistudio.google.com/apikey → Create API Key → copy it. This powers Siya's brain.
Open AI Studio
2
Paste key in Integrations
Go to Integrations → Gemini API key and paste it. Keys are saved locally in your browser.
3
Open Call logs → Call Siya
Click Call Siya now — free. Chrome will ask for microphone permission — click Allow.
4
Speak naturally
Siya will greet you in Hinglish, then ask qualification questions. You respond by voice.
5
(Optional) Connect Google Sheets
Add your Sheets webhook URL in Integrations to auto-log every call with transcript + score.
POST → your webhook → JSON payload
6
(Optional) Edit Siya's prompt
Go to Agent config to customise Siya's opening line, qualification questions, and tone.
7
Use Chrome or Edge only
The SpeechRecognition API used for STT is only available in Chromium-based browsers. Safari will not work.
